## How to run at you local-
1. Download or clone project
2. `sudo apt-get install python3-dev libpcap-dev pypcap`
3. pip install -r requirements.txt
4. sudo su
5. python app.py

## Dependencies:
    1. python3-dev  
    2. libpcap-dev  
    3. pypcap
